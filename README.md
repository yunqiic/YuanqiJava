# YuanqiJava
Yuanqi Java SDK

### 使用SDK
```java
// spring boot, inject agentService
ChatRet result = this.agentService.chat(params);
```

### maven
### Maven 引用方式
```xml
<dependency>
    <groupId>com.github.zhangchunsheng</groupId>
    <artifactId>yuanqi-java-agent</artifactId>
    <version>1.0.2</version>
</dependency>
```
