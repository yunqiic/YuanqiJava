# 使用说明
1. 在自己的Spring Boot项目里，引入maven依赖
```xml
    <dependency>
        <groupId>com.github.zhangchunsheng</groupId>
        <artifactId>yuanqi-java-agent-spring-boot-starter</artifactId>
        <version>${version}</version>
    </dependency>
 ```
2. 添加配置(application.yml)
```yml
yuanqi:
  key:
```
